Flow2API Worker

Supplies real-browser reCAPTCHA tokens + keeps the Google Labs login fresh. Keep a Labs tab logged in.

Base URL only — the captcha socket and token endpoint are derived from it.

Must match this account's token's extension_route_key in the admin panel. Leave empty if the account's token has no route key.

Higher = slower minting but fewer TOO_MUCH_TRAFFIC rate flags under heavy load. 0 = none. ~2000 ≈ ≤30/min.

ON (default): this profile auto-gets its OWN static IP (one of the proxy's IPs), so it mints reCAPTCHA from a different IP than your other profiles — zero setup. Only the Flow site + reCAPTCHA URLs are proxied; your normal browsing (Google search, everything else) and the backend stay direct. If a profile gets a Google "verify it's you", re-login through it (or turn this off to revert).

Leave blank to use Auto above. Set a URL only to pin a specific proxy/session for this profile.
